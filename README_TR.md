## Bağımlılıklar
Debian tabanı için: `apt install ffmpeg` <br/>
Arch tabanı için: `pacman -S ffmpeg4.0`

## Kurulum
```sh
git clone https://github.com/beucismis/r2tg_bot
```
praw.ini ve pyrogram.ini dosyalarını düzenle.
